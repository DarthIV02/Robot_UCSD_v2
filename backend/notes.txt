** Created Python virtual environment to install Flask

https://python.land/virtual-environments/virtualenv
python -m venv <directory>
python -m venv venv

** Activate virtual environment

https://python.land/virtual-environments/virtualenv
source venv/bin/activate

** Install mongob community (MacOS)

Pre-requisite: install brew package
brew tap mongodb/brew
brew update
brew install mongodb-community@7.0 (check official website for latest version available https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/)

** Run mongodb (installed with brew MacOS)

brew services start mongodb-community
To verify mongodb was starred: brew services list

** Connect and use mongodb
Open new terminal and run: mongosh

** Stop mongodb (installed with brew MacOS)

brew services stop mongodb-community
